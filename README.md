# 🌸 PCOS AI Health Companion

> An AI-powered wellness platform designed to help women manage PCOS with personalized insights, tracking, and smart recommendations


---

## 🚀 Features

✨ **Personalized Health Tracking**

* Track menstrual cycles and symptoms
* Monitor overall wellness patterns

🥗 **Smart Diet Plans**

* AI-based diet recommendations
* PCOS-friendly meal suggestions

📊 **Health Analytics**

* Visual insights into user health trends
* Data-driven recommendations

🤖 **AI Chatbot**

* Get instant health guidance
* Ask PCOS-related questions anytime

🏥 **Healthcare Support**

* Hospital and doctor recommendations
* Easy access to resources

---

## 🧠 Tech Stack

* **Frontend:** React / Vite / TypeScript
* **Styling:** Tailwind CSS
* **AI Integration:** OpenAI / Custom logic
* **Deployment:** Vercel

---

## 🎯 Key Highlights

* Clean and modern UI (Glassmorphism + Gradient design)
* AI-powered recommendations
* User-centric design for real-world impact
* Scalable architecture

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/pcos-ai-health-companion.git

# Navigate to project
cd pcos-ai-health-companion

# Install dependencies
pnpm install

# Run project
pnpm run dev
```
